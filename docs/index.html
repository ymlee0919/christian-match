Juegos

Cara a cara 

Juego simple de responder rondas de preguntas, una cada uno, hasta que se agoten las preguntas o hasta que decidan.
El objetivo es ver cuál de los dos alcanza mayor puntuación.

El rápido

La idea es ver cuál de los dos responde una pregunta más rápido. El primero que responda correctamente es el que gana los puntos. Esta competencia depende de la sinceridad de los jugadores pues se tiene que indicar manualmente cuál fue el jugador ganador.

Escalando

El objetivo es ir escalando hasta ver quién llega primero a la cima y alcanza la corona. Los jugadores deben seleccionar la cantidad de pasos que deseen dar. Según ascienden los jugadores aumenta el nivel de complejidad de las preguntas. Si un jugador no responde una pregunta correctamente retrocede la misma cantidad de pasos que decidió avanzar.

Baseball

Clásico juego de baseball bíblico entre dos equipos.

¿Cómo empezar?

  1. Abre la aplicación: Sólo tienes que ejecutar el archivo ChrisitanMatch.exe.
  2. Definir equipos: Una vez abierta la aplicación el siguiente paso es "Definir Equipos". Al oprimir el botón se muestra una ventana que permite seleccionar los personaje bíblicos con los que se llamarán los equipos. El primer jugador se selecciona con clic izquierdo y el segundo jugador con clic derecho.
  3. Seleccionar juego: Selecciona el tipo de enfrentamiento que desean jugar.
  4. Ajusta los detalles del juego: Antes de comenzar cada juego es necesario establecer ciertas reglas del juego.
  5. A jugar...!!!
Requerimientos

La aplicación sólo corre sobre Windows con Framework de .NET 3.5 o mayor.

Configurando las partidas

Antes de comenzar cada juego es necesario establecer ciertas reglas del juego.
Esto es lo que significa cada posible configuración:

Mostrar cuestionario con posibles respuestas

Si se selecciona esta opción se muestran 5 posibles respuestas cuando se lanza una pregunta.

En caso de no seleccionarla el jugador debe responder y después se revela la respuesta correcta. La respuesta se muestra cuando hace clic en “Respuesta”, que aparece en la misma ventada donde se hace la pregunta.

Limitar el tiempo de respuesta

Esto es para limitar el tiempo que puede demorar un jugador en responder. Si no se selecciona el tiempo para responder una pregunta es ilimatado. Si lo selecciona se debe ajustar el tiempo permitido para dar la respuesta. Una vez pasado este tiempo sin seleccionar alguna respuesta se considera que la respuesta es incorrecta.

Todas las preguntas tienen el mismo valor

Establece el valor de las preguntas, todas por igual, sin tener en cuenta el grado de complejidad. La dificultad de una pregunta se puede ver en cuestionario y se indica con asteriscos debajo de la pregunta, en color amarillo. El grado de dificultad puede estar entre 1 (más fácil) y 5 (más difícil).

Cuestionarios

Es posible crear o personalizar cuestionarios. Los cuestionarios son un compendio de preguntas. Cada pregunta tiene el nivel de dificultad (entre fácil y difícil), el texto de la pregunta, 5 posibles respuestas y un comentario relativo a la pregunta correcta. Este es el comentario que se muestra cuando un competidor selecciona una respuesta incorrecta.

La aplicación permite gestionar los cuestionarios. Una vez creado un nuevo cuestionario o abierto uno ya existene, es posible comenzar a editar las preguntas.
